## How to install

```bash
  # Clone the project
  git clone https://github.com/whosramoss/watch-dogs-2-game-soundtrack/

  # Go to the project directory
  cd watch-dogs-2-game-soundtrack

  # Install dependencies
  npm install

  # Start the server 
  npm run dev
```
